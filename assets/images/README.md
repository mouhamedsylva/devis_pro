# Assets Images

## Logo de l'application

Placez votre fichier `logo2.png` dans ce dossier.

### Spécifications recommandées :
- **Format** : PNG avec transparence
- **Taille** : 512x512 pixels (minimum 200x200)
- **Ratio** : 1:1 (carré)
- **Poids** : < 500 KB

### Utilisation :
Le logo est utilisé dans le splash screen avec animations au démarrage de l'application.

Si vous n'avez pas encore de logo, l'application utilisera une icône par défaut (receipt_long_rounded).
